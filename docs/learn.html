Activity Cards

Activity cards present simple scripts to help users get started with an activity. They are designed to be printed two per page and folded in half to make a two-sided card. To make them last longer, the cards can be laminated or printed on card stock.

micro:bit

These cards help students get started with the micro:bit. No additional components are needed except where noted.

  • Getting Started
  • Sensors
  • Sound
  • Radio
  • NeoPixels (requires NeoPixels)
  • Servos (requires a servo and a way to connect it)
  • IR RemoteControl (requires an IR receiver and an IR remote control)

Circuit Playground Express

These cards help students get started with AdaFruit Circuit Playground Express. No additional components are needed except where noted.

  • Getting Started
  • NeoPixels
  • Sound
  • IR Remote Control (requires an IR remote control)

Mozilla IoT

These cards show how to connect MicroBlocks projects to the internet using Mozilla's free, open-source WebThing Gateway software running on a Raspberry Pi. This allows users to interact securely with MicroBlocks projects using any web browser–even from their phone!

  • Web Light video
  • Web Button video
  • Run-in-place Race video
  • Hello LED
  • Doorbell (NeoPixels optional)
  • WebThings Gateway

Guided Activities

A guided activity is a series of experiments that explore a set of related ideas. While these activity guides are written in the form of "teacher notes" they can also be used as self-guided tutorials by motivated learners.

  • Getting Started with MicroBlocks
  • Color Mixing
  • Exploring Electricity
  • Make a micro:bit Dance
  • Time Scales
